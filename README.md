# Overcharged

A small Unity game that teaches energy efficiency by challenging players to turn off lights and appliances to lower an electricity bill. Built for Knight Hacks VII (3rd Place â€” Next Era: Energy Efficiency Challenge).

Overview
- Fast-paced, arcade-style gameplay inspired by Overcooked and the NextEra challenge.
- Players flip switches and interact with devices to reduce energy waste and earn a better score.

Tech used
- Unity (game client, scenes, prefabs)
- C# (game logic and backend)
- MongoDB (cloud database)
- ShaderLab / HLSL (visual polish)

Highlights
- Won 3rd Place in the Next Era: Energy Efficiency Challenge at Knight Hacks VII.
- Built login and leaderboard pages to enhance user engagement.
- Designed four C# endpoints to connect Unity with MongoDB.
